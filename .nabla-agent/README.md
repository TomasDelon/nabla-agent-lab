# .nabla-agent

This directory is the lab message bus for agent orchestration experiments.

## Directories

| Path | Purpose |
|---|---|
| `prompts/` | Human- or planner-written task prompts |
| `runs/` | Immutable run reports generated by workflows |
| `reviews/` | Optional read-only review summaries |
| `state.json` | Current lab state and safety flags |

## Write Policy

GitHub Actions may write run reports and PR branches.

ChatGPT scheduled tasks must be read-only. They may inspect this directory and summarize findings, but they must not create or update files during scheduled execution.

## Run Report Contract

Each run should create a directory under `runs/` containing at least:

- `prompt.md`
- `status.json`

Depending on the workflow, it may also contain:

- `output.md`
- `proposal.md`
- `context.md`

## Safety Flags

`state.json` contains human-readable safety state. The important flags are:

- `writerAllowed`
- `autoMergeAllowed`

For production-style workflows, `autoMergeAllowed` should remain `false`.
